🚀 PUBLISHING STEPS
─────────────────────────────────────────────────────────────────────────────

1. ADD ICON
   └─ Create 128x128 PNG and save as script-runner/icon.png

2. TEST LOCALLY
   └─ cd script-runner
   └─ npm install
   └─ Press F5 in VS Code

3. CREATE MARKETPLACE ACCOUNT
   └─ Go to https://marketplace.visualstudio.com
   └─ Create publisher: "SanekxArcs"

4. GET PERSONAL ACCESS TOKEN
   └─ Go to https://dev.azure.com
   └─ Create PAT with "Marketplace: Publish" scope

5. INSTALL VSCE
   └─ npm install -g vsce

6. PUBLISH
   └─ cd script-runner
   └─ vsce publish --pat YOUR_TOKEN
